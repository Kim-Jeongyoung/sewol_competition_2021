유의사항

1명당 분야별 1점 제출 가능하며 중복 응모는 불가능함. 응모 작품은 반드시 제시된 규격에 맞아야 하며, 규격과 다르게 응모할 시에 심사 대상에서 제외될 수 있음 응모 작품은 기존에 응모, 발표되지 않은 신작이어야 하며, 순수한 창작물이 아닌 모방, 표절, 대리 작성된 작품으로 밝혀질 경우 심사에 제외되며, 수상 이후에라도 이 같은 사실이 밝혀질 경우 수상이 취소됨 응모 작품은 저작권에 문제가 없어야 하며 심사·선정 이후 표절, 복제 등으로 저작권을 침해한 사실이 확인 될 경우 심사에 제외되며, 수상하였더라도 사후 수상이 취소 됨. 분쟁 발생 시 응모자 책임을 전적으로 부담함 응모 된 모든 작품은 수상여부와 관계없이 반환하지 않으며, 출판·전시·교육·홍보 등 공익적 목적으로 이용할 수 있음. 이 경우 수상작을 있는 그대로 이용하는 것 이외 2차적 저작물로 작성하여 이용하는 것을 포함 함 타인 명의의 응모 접수는 불가하며 반드시 본명으로 응모해야 함 심사는 각 분야별 전문가들로 비공개로 진행되며, 당선작이 없다고 판단될 경우 당선작을 선정하지 않을 수 있음
